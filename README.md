## Clock-App

### 실행방법
1. `npm install lite-server -g`
2. `lite-server` 실행
3. `localhost:3000`, terminal에 표시된 `PORT` 번호 확인 후 접속

### 추가구현 해볼만한 사항
- [ ] 일괄삭제/선택삭제 버튼
- [ ] active 알람 깜빡이게
